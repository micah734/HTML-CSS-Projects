Article Tag

is a HTML tag that specifies independent, self contained content. Potential uses are Forum posts, blog post and news stories

Details Tag

The details tag specifies additional details that the user can open and close on demand. They are often used to make interactive widgets. Summary tag is used in conjunction with details to specify a visible heading for the details.
